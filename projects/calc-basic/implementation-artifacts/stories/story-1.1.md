# Story 1.1: Build Calculator UI Shell

**Priority**: High  
**Complexity**: Medium  
**Est. Time**: 5 minutes

## Description
Create HTML structure and brutalist CSS foundation for calculator.

## Acceptance Criteria
- [ ] HTML5 structure with JetBrains Mono font from Google Fonts
- [ ] CSS custom properties for black/white/red color scheme
- [ ] Calculator container: 420px × 640px, 4px black border, centered
- [ ] Display: 120px height, black bg, white text, right-aligned
- [ ] Button grid: 4×5 layout, 5px gaps, all 20 buttons (0-9, +, -, ×, ÷, =, C, .)
- [ ] Button styling: 2px black borders, white bg, hover invert
- [ ] Zero spans 2 columns, equals spans 2 rows
- [ ] Responsive on mobile (<600px)
